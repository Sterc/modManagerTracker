# modManagerTracker
Track the MODX Manager using Hotjar or Visual Website Optimizer (VWO).

## Usage
1. Create a new plugin
2. Paste the code from modManagerTracker.plugin.php
3. Enable the event ```OnManagerPageBeforeRender```
